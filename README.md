# 🚀 AI Research Assistant

An AI-powered Research Assistant built using **Spring Boot + Gemini API** that helps users summarize content and get intelligent suggestions instantly.

---

## 💡 Features

- 📄 Summarizes long text into concise insights
- 🧠 Suggests related topics and further reading
- ⚡ Fast AI response using Gemini API
- 🔌 REST API based backend
- 🌐 Deployed on cloud

---

## 🌐 Live API

🔗 https://research-assistant-xs2p.onrender.com/api/research/process

---

## 🛠️ Tech Stack

### Backend:
- Java
- Spring Boot
- REST APIs
- Gemini API (Google AI)

### Tools:
- Postman (API testing)
- Render (Deployment)

---

## ⚙️ How It Works

1. User sends content + operation (summarize / suggest)
2. Backend builds prompt
3. Request sent to Gemini API
4. AI generates response
5. Response parsed and returned to user
